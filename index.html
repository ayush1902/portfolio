Cancer Detection and Machine Learning

Research

Machine Learning

Supervised Learning

Problem

Breast cancer continues to afflict people with irreversible damage. However, if diagnosed early, it is found to be treatable in most of the scenarios. Can a machine learning model help hospitals model patients better in detecting cancer early?

Solution

Using supervised machine learning techniques on breast cancer data set consisting of tumour information in order to classify cancer as bening or malignant. Furthermore, predicting recurrance of the same in patients diagnosed with one.

Impact

Why limit to breast cancer? Extending the functionality to four major types of cancer and evaluating and hypothesisng a general rule of thumb while evaluating models.

Design interfaces for Autism

Research

Accessibility

User Research

UX Design

Problem

There is no one-size-fits all approach for interface design, especially when the target audience is differently-abled. How might we aprpoach the design of learning application that feels more human to autistic children?

Solution

An application design process that uses gamification and includes the user at every stage; from intervirews, to contextual inquiries, to observations and even feedback on wireframes/prototypes.

Impact

The game metaphor proved to be a winning strategy amongst the children, bolstering engagement. However, there were some key outcomes pertaining to teachers and the limitations of the topics that could be taught.
